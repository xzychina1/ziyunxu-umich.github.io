Case study

Aurora

A landing page concept for a modern productivity subscription. The goal was to make the product feel premium, trustworthy, and easy to sign up for in under one minute.

Back to portfolio Next project

Challenge

The main challenge was balancing motion and visual richness with a simple conversion path. The page needed to communicate value quickly without feeling crowded.

Solution

I used a focused hero section, strong hierarchy, compact feature cards, and a single primary call to action repeated at key points on the page.

Role and tools

Front-end layout, visual direction, content structure, and responsive design using HTML and CSS.
